Discord Bot

This Discord bot tracks "combos" based on user interactions. When different users post the same text consecutively, the bot increments and displays a "combo" count. If the same user repeats a comment or the text changes, the combo breaks, and the final length is announced. Initially hosted on Replit, it was later migrated to an Ubuntu cloud instance for improved stability. The migration included moving the old database and setting up a new SQLite3 database, with three additional files created for database management. The GitHub version has been sanitized to remove sensitive information. This project was a valuable learning experience, integrating various technologies such as Discord.py, cloud management, Ubuntu and SQLite3. As of now, the bot is active on nearly 200 Discord servers, showcasing its stability and user engagement.
